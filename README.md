This document is a **comprehensive compilation of controversies, criticisms, and concerns** about Donald Trump raised by Democrats, progressives, moderates, and some Republicans. It is **explicitly one-sided** and does not attempt to present a balanced view of Trump's presidency.

- Documentation of statements, actions, and policies that critics find problematic
- A resource for understanding opposition perspectives

**What this document IS NOT:**
- A balanced assessment of Trump's presidency
- A list of policy accomplishments or successes
- An objective historical analysis
- A legal brief or definitive fact-finding report
