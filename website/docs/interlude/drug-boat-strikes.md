# Drug Boat Strikes Save 25,000 Lives Each (2025)

- Claimed each boat strike in the Caribbean saves **25,000 lives** from drug overdoses
- Johns Hopkins professor called the claim **"absurd"** [Source: Johns Hopkins expert statement, 2025; U.S. News]
- No evidence any boats were carrying drugs [Source: Washington Post investigation, 2025]
